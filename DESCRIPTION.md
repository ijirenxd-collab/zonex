# Bunzo Astro CMS - Premium Edition

Este é um template de blog de alta performance desenvolvido em **Astro 5** com **Tailwind CSS**, ideal para sites que buscam velocidade extrema, SEO amigável e facilidade de gerenciamento.

## 💎 Destaques do Template

- ⚡ **Performance Imbatível**: Graças ao Astro, o template entrega o carregamento mais rápido do mercado, garantindo ótimas posições no Google.
- 🎨 **Customização Dinâmica**: Pela primeira vez, um template Astro que permite trocar **Cores Primárias/Accents** e **Pares de Fontes** diretamente pelo Painel Administrativo, sem tocar no código.
- 📈 **Focado em Leads**: Sistema de captura de newsletter integrado ao plugin de 'Lista de Emails' e formulário de contato com painel dedicado para gestão de leads.
- 🏷️ **Nicho de Afiliados**: Design e componentes (Hero Slider, Cards de Reviews, Sidebar) otimizados para blogs de recomendação de produtos e reviews técnicos.
- 🛠️ **CMS completo**: Gerenciamento de Posts, Categorias, Autores, Menus e Páginas Institucionais (Sobre, Contato, Políticas).

## 🛠️ Tecnologias Utilizadas
- **Astro 5**: Framework principal para máxima performance.
- **Tailwind CSS**: Estilização moderna e responsiva.
- **React**: Usado no painel administrativo e ilhas de interatividade.
- **JSON-based Data**: Armazenamento simples, rápido e fácil de migrar.
